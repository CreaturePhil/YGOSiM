## Getting Started

First, start up MongoDB:

```bash
$ mongod
```

Then, open up another terminal window:

```bash
$ git clone https://github.com/stevoduhhero/YGOSiM.git && cd YGOSiM
$ npm install
$ node lib/server.js
```
